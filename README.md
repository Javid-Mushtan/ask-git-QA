# 🚀 Ask-Git

Ask-Git is a FastAPI-based backend service that allows users to interact with GitHub repositories using natural language queries. It processes repository data and returns structured insights about code, files, and project logic.

---

## 📌 Features

- 🔍 Analyze GitHub repositories
- 🤖 Natural language query support
- 📁 Extract file and project structure
- ⚡ FastAPI high-performance backend
- 🐳 Dockerized for easy deployment
- 🔐 Environment variable support (.env)

---

## 🏗️ Tech Stack

- Python 3.10+
- FastAPI
- Uvicorn
- GitPython
- Docker
- dotenv
